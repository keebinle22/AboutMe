## Overview
**Instagram KO** is a full-stack web application created to showcase my development skills. Using a combination of **React** for the frontend, Bootstrap for responsive design, **Spring Boot** for the backend, and **MongoDB** for database management, this project mirrors many of the core features found in Instagram. It serves as a demonstration of my ability to integrate various technologies to build a functional and user-friendly application.
